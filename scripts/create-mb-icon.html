MB App Icon Generator

Generate "MB" app icons for Android and iOS

Instructions:

  1. Click "Generate All Icons" to create previews
  2. Right-click on each icon and "Save Image As..."
  3. For Android: Save icons as:
    • mdpi: 48x48 → ic_launcher.png
    • hdpi: 72x72 → ic_launcher.png
    • xhdpi: 96x96 → ic_launcher.png
    • xxhdpi: 144x144 → ic_launcher.png
    • xxxhdpi: 192x192 → ic_launcher.png
  4. For iOS: Use the 1024x1024 icon and upload to AppIcon.appiconset in Xcode
